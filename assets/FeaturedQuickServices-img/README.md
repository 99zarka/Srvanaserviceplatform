# Featured Quick Services Images

This directory contains images for the Featured Quick Services section.

## Required Images

Place the following images in this directory:
- `ac-cleaning.webp` - Air conditioning/HVAC services
- `plumbing-leaks.webp` - Plumbing services
- `electrical-fixing.webp` - Electrical services
- `painting.webp` - Painting/carpentry services

## Note

The component currently uses Unsplash placeholder images. Replace them with your actual service images for production.
